Research on Building Deep Learning Models for Non-Experts

Academic Research • Machine Learning • HCI

machine learning HCI research academic accessibility

Overview

Conducted comprehensive research on making deep learning model development accessible to non-experts. This interdisciplinary study combined human-computer interaction (HCI) principles with machine learning to create intuitive interfaces and workflows that enable domain experts to leverage AI without extensive technical knowledge.

Research Question

How can we design user interfaces and interaction paradigms that allow domain experts (without ML expertise) to effectively build, train, and deploy deep learning models for their specific use cases?

Methodology

The research employed a mixed-methods approach:

  • User studies with 50+ domain experts across various industries
  • Prototype development and iterative testing
  • Usability testing and cognitive load analysis
  • Longitudinal studies on model adoption and success rates
  • Comparative analysis of existing no-code ML platforms

Key Findings

Visual Programming

Domain experts preferred visual, node-based interfaces over code-based approaches for model configuration.

Contextual Guidance

Contextual help and guided workflows significantly improved success rates and user confidence.

Iterative Feedback

Real-time feedback and model performance visualization were crucial for user understanding.

Domain-Specific Templates

Pre-built templates tailored to specific domains reduced cognitive load and improved outcomes.

Design Principles

Based on the research findings, I developed a set of design principles for non-expert ML interfaces:

  1. Progressive Disclosure: Reveal complexity gradually as users become more comfortable
  2. Domain Translation: Use domain-specific language and metaphors
  3. Immediate Feedback: Provide real-time validation and performance indicators
  4. Guided Exploration: Offer structured paths while allowing customization
  5. Error Recovery: Design forgiving interfaces that help users learn from mistakes
